## Getting Started

1. Clone this repository
2. Required items to be installed:
- [node.js](https://nodejs.org) (version >= 5.3)
- [npm](https://www.npmjs.com)
- [mocha](https://mochajs.org)
- [gulp](http://gulpjs.com/)
3. ```$ npm install```
